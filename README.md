# Iris Dataset - Exploratory Data Analysis (EDA)

## Overview
Exploratory Data Analysis on the classic Iris dataset using Python.

## Dataset
- Source: Kaggle (Iris Dataset by Himanshu Nakrani)
- 150 rows, 5 columns
- 3 species: Setosa, Versicolor, Virginica

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## What I Did
- Loaded and explored the dataset
- Checked basic statistics using describe()
- Visualized species distribution using pairplot and boxplot

## Key Findings
- Setosa is the most distinct species and easiest to separate
- Petal length and petal width are the most useful features to differentiate species
